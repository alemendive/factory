SGI
Sistema de gestion de Inventario


| Sobre |


SGI es un sencillo software de gestión de inventario creado para las necesidades de una empresa. El software es capaz de realizar un seguimiento del stock entrante y saliente con una factura renderizada dinámicamente. A cada proveedor se le proporciona un perfil donde puede ver el historial de sus transacciones con la empresa.


| Características |


  • Gráfico interactivo para mostrar el stock disponible actual
  • Se mantienen los detalles del stock, como el stock disponible actual.
  • Las transacciones entrantes se realizan a través de compras.
  • Las transacciones salientes se realizan a través de ventas.
  • Cada transacción realizada a través del software se registra
  • La factura se representa dinámicamente y se puede imprimir.
  • Se mantienen perfiles separados para proveedores.
